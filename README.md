# 数据分析与数据产品作品集

这是一个用于 GitHub Pages 的静态作品集目录。

## 页面结构

- `index.html`：作品集首页
- `cases/push.html`：Push 触达关闭率分析与优化
- `cases/bi-platform.html`：指标体系、主题报表与自助分析平台
- `cases/data-agent.html`：数据 Agent 分析助手
- `cases/data-governance.html`：大数据成本治理
- `dashboards/app-store-dashboard.html`：大盘概况经营驾驶舱 Demo
- `dashboards/cost-dashboard.html`：数据成本治理看板 Demo

## 脱敏说明

公开页面中的业务场景、图表、数据和指标均经过抽象和模拟处理，不展示真实公司名称、系统名称、表名、点位名称、内部截图或真实经营数据。
